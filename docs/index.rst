Social Network Kata Documentation
=================================

A small, Docker-packaged social-network backend kata built test-first across
three bounded contexts — Users, Posts/Timeline, and Messaging. This site
collects the architecture documentation (arc42) and the user-story bundles
that drive the TDD/BDD cycle.
